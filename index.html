    • What is Typescript?

    Typescript is a superset of JavaScript which primarily provides optional static typing, classes and interfaces.

    Why Typescript and not Javascript?

    Typescript attempts to solve some of the issues with Javascript
    Javascript was never really designed to be programming language for big applications
    Typescript is a static typed Language which performs type Checking at runtime
    JavaScript is Dynamically typed - Types checked on the fly, during execution
    Note: - I am not saying don't learn JavaScript here

    Typescript Types!

  • Number: number (represents both Integer as well as Floating-Point numbers)
    1028
  • Boolean: boolean (Represents true and false)
    true
  • String: string (It is used to represent a sequence of characters)
    "Hello, world!"
  • Any: any (variable is declared with any data-type then any type of value can be assigned to that variable)
    any
  • Void: void (Generally used onfunction return-types)
    void
  • Null: null (It is used when an object does not have any value)
    null
  • Undefined: undefined (Denotes value given to uninitialized variable)
    • undefined
